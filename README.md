## Quick Start ⚡

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # lib/build/
npm run preview
```

**Note**: Uses standard GSAP (no club/trial plugins). Deploy-ready!

## Deployment 🚀
- **Vercel**: `vercel --prod` (vercel.json configured)
- **Netlify**: Drag `dist/` folder
- Analytics: @vercel/analytics included
